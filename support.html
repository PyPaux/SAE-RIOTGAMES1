Comment pouvons-nous vous aider ?

Contacts

Pour tous problèmes,vous pouvez nous contacter par mail à l'adresse suivante :
support-covoiturage@riotgames.com

FAQ

Comment puis-je proposer un trajet ?

Vous devez vous connecter à votre compte pour pouvoir proposer un trajet. Une fois connecté, vous pouvez cliquer sur le bouton "Proposer un trajet" dans la barre de navigation.

Comment puis-je rechercher un trajet ?

Vous devez vous connecter à votre compte pour pouvoir rechercher un trajet. Une fois connecté, vous pouvez cliquer sur le bouton "Rechercher un trajet" dans la barre de navigation.

Comment puis-je louer un véhicule ?

Vous devez vous connecter à votre compte pour pouvoir louer un véhicule. Une fois connecté, vous pouvez cliquer sur le bouton "Louer un véhicule" dans la barre de navigation.

Comment puis-je me connecter ?

Vous pouvez vous connecter en cliquant sur le bouton "Connexion" dans la barre de navigation. Vous pouvez également cliquer sur le bouton "Connexion" dans la page d'accueil.

Comment puis-je me créer un compte ?

Si c'est la première fois. Vous pouvez vous créer un compte en envoyant un mail à support-covoiturage@riotgames.com

Je n'arrive pas à me connecter

L'enregistrement au programme de covoiturage n'est pas automatique. Pour vous faire enregistrer où pour tout autre problème, veuillez envoyer un mail à support-covoiturage@riotgames.com .

Comment puis-je supprimer mon compte ?

Vous pouvez demander une suppression de votre compte en envoyant un mail à support-covoiturage@riotgames.com
